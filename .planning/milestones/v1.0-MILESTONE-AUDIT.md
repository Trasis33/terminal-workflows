### Phase 3: Management TUI

- **Help text mismatch**: `cmd/wf/manage.go` help text references `ctrl+t` for settings, but actual keybinding is `S` (shift-s). Cosmetic only — the TUI's own hints bar shows the correct key.

### Phase 5: AI Integration

- **Stale test expectations**: 2 tests in `internal/ai/copilot_test.go` fail due to outdated model name constants:
  - `TestModelConfigForTask` "all empty falls to default" expects `gpt-4o-mini` but `ForTask()` returns `gpt-4.1`
  - `TestDefaultModelConfig` expects `gpt-4o`/`gpt-4o-mini` but `models.go` defaults are `gpt-5-mini`/`gpt-4.1`
  - Fix: Update test expectations to match current `models.go` defaults

### Total: 2 items across 2 phases

Neither item affects runtime functionality. Both are documentation/test maintenance tasks.
